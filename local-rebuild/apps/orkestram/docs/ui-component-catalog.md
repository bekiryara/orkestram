# UI Component Catalog (Bootstrap Bazlı)

Bu katalog, ajanların ve geliştiricilerin ortak sınıf dilini kullanması içindir.

## 1) Button
- Primary: `btn btn-primary`
- Secondary: `btn btn-outline-secondary`
- Danger: `btn btn-outline-danger`

## 2) Card
- Kapsayıcı: `card`
- İçerik: `card-body`
- Başlık: `card-title`

## 3) Input / Label
- Label: `form-label`
- Input: `form-control`
- Select: `form-select`
- Yardım metni: `form-text`

## 4) Table
- Temel: `table`
- Çizgili: `table table-striped`
- Responsive: `table-responsive`

## 5) Tabs
- Sekme listesi: `nav nav-pills`
- Sekme öğesi: `nav-link`
- Aktif öğe: `nav-link active`

## 6) Badge
- Durum etiketi: `badge text-bg-primary|secondary|success|warning|danger`

## 7) Alert
- Bilgi: `alert alert-info`
- Başarı: `alert alert-success`
- Hata: `alert alert-danger`

## 8) Section Title
- Başlık: `h2` + `mb-3`
- Alt metin: `text-muted`

## 9) Layout Yardımcıları
- Grid: `row`, `col-*`
- Aralık: `mt-*`, `mb-*`, `p-*`, `gap-*`
- Hizalama: `d-flex`, `justify-content-between`, `align-items-center`

## Kullanım Kuralı
Yeni ekran/alan geliştirirken önce bu katalogdan class seçilir. Karşılık yoksa önce katalog genişletilir, sonra kod yazılır.
