# Decision 0001 - Izole Kategori Calisma Modeli

Tarih: 2026-03-09

Karar:
1. Kategori sistemi calismasi tamamen `docs/category-workspace` altinda ilerler.
2. Ana repo plan/durum belgelerine simdilik dokunulmaz.
3. Hedef, Armut setinden sitemize uyan tum kategorileri deterministik sekilde bulmaktir.
